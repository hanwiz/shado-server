This page is currently under construction, please check back soon! Welcome to the Simulator of Humans and Automation in Dispatch Operations (SHADO)! This tool, designed by Duke University researchers, simulates the workload of a dispatch operator and associated freight rail operators across the duration of a given shift. With SHADO, you can choose a trip with unique conditions and then see the operators' average workload after thousands of similar shifts.

Get Started

If you're ready to get started, then run the simulator! And if you encounter any issues or have questions about the simulation, please contact us!
